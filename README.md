# Secure Web Server on AWS using VPC, EC2, and Security Groups

## Project Overview
This project demonstrates how to deploy a secure web server on AWS using core cloud services. I created a custom VPC, configured networking components, launched an EC2 instance, installed Apache, and hosted a simple webpage.

## Services Used
- AWS VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group
- EC2
- Apache Web Server

## Architecture
User -> Internet Gateway -> VPC -> Public Subnet -> EC2 Instance -> Apache Web Server

## What I Did
- Created a custom VPC
- Created a public subnet
- Attached an Internet Gateway
- Configured route table for internet access
- Set security group rules for HTTP and SSH
- Launched an EC2 instance
- Installed and started Apache
- Hosted a test webpage

## Outcome
The web server was successfully deployed and accessible through the EC2 public IP in a browser.
